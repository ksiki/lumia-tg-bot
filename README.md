# Lumia Telegram Bot

This project is a Telegram bot built with Python.  
It uses the aiogram framework to handle messages and user actions.

## Technologies used

- **aiogram**
- **PosgreSQL**
- **Docker Compose**
- **Git Actions**

## About the Project

The bot gives users different predictions and reports.  
It can work with tarot cards, astrology, and other spiritual topics.

The bot also can:
- send messages with templates
- generate PDF reports
- save user data in a database
- manage subscriptions and payments

## Main Parts

- **bot/** – main logic of the bot  
- **database/** – work with database and data models  
- **scenarios/** – user flows and bot behavior  
- **assets/** – templates, texts, and files  
- **utils/** – helper functions  
- **tests/** – unit tests  

## Features

- Telegram bot with menu and commands  
- Prediction system (tarot, compatibility, etc.)  
- PDF generation from templates  
- Database with users and transactions  
- Logging and middleware support  

## Purpose

This project shows how to build a complex Telegram bot  
with clean structure and many features.
